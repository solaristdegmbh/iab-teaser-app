IAB Teaser Web-App

Teilbare Online-Version ohne Server

Diese Version laeuft komplett im Browser. Du fuellst die Daten aus, siehst sofort die Vorschau und erzeugst die PDF direkt lokal im Browser. Dadurch kann die App als statische GitHub-Pages-Seite veroeffentlicht werden.

Ablauf

Beispiel laden, Felder anpassen, Bilder hochladen und auf PDF exportieren klicken. Die Eingaben bleiben auf Wunsch lokal im Browser gespeichert.

  • keine Python-Installation beim Endnutzer
  • kein Server notwendig
  • perfekt fuer GitHub Pages
